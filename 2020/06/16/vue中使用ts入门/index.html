为什么使用typescript

  • 支持所有原生javscript的语法
  • 类型检查。使js具有强类型语言的特性
  • vscode支持友好
  • 微软开源，各大框架源码等皆开始使用ts开发

vue中使用typescript

  • vue create ts-demo时选择Typescript、class-style的语法
  • 下载vue-class-component、vue-property-decorator插件
