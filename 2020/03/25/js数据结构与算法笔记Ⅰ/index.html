# 栈是一种遵从先进后出原则的有序结合。

新添加的或待删除的元素都保存在栈的同一端，为栈顶，另一端则为栈底。且新元素都靠近栈顶，旧元素都接近栈底。
例：一摞书、堆起的盘子
